TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════

Problem: "Site can't be reached" or "localhost not reachable"
Solution: 
  1. Make sure backend is running first (check start_backend.bat)
  2. Look for "placement_tracker.db" in backend folder
  3. Backend terminal should show "Application startup complete"

Problem: "Module not found" errors
Solution:
  - Backend: Run: cd backend && pip install -r requirements.txt
  - Frontend: Run: cd frontend && npm install

Problem: "Port already in use"
Solution:
  - Change port in backend/run.py (line 13)
  - Or close other apps using port 8000
